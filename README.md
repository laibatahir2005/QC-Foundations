 Quantum Computing Foundations

A beginner-friendly collection of Python programs covering the fundamental concepts required for quantum computing.

 Topics Covered

* Complex numbers and vectors
* Dirac (Bra-Ket) notation
* Probability and state normalization
* Coin-flipping simulations
* Probabilistic states
* Basic quantum-mechanics concepts used in quantum computing
* Introduction to quantum state representation

 Goal

The purpose of this repository is to build intuition for quantum computing by connecting mathematical concepts, probability theory, and Python programming before moving to quantum circuits and algorithms.
